# Ngrok-Tool
__________________________________________________________
Easy Ngrok Port Forwarding Tool!
__________________________________________________________
---How to install---

1. Download "Ngrok-Tool.zip"
2. Unzip "Ngrok-Tool.zip"
3. Open "Ngrok-Tool"
4. Unzip "Ngrok.zip"
5. Start Ngrok-Tool.bat
